# Validação Objetiva de Dados Sintéticos por Paráfrase T5 em HRI
Criador: Iraê César Brandão  - Desenvolvedor e Programador
Disponibilizado em: <https://irae-cesar-brandao.github.io/Valid_Abord_Parafrase_T5/>


## Contexto Acadêmico

Este repositório foi criado **exclusivamente para fins acadêmicos, educacionais e de estudo analítico**.  
O projeto não tem caráter comercial nem representa um sistema de produção, sendo voltado à **análise metodológica**, **experimentação controlada** e **exploração conceitual** de técnicas de validação de dados sintéticos em Processamento de Linguagem Natural (NLP).

O conteúdo aqui apresentado serve como **material de apoio didático**, **protótipo experimental** e **base para estudos em pós-graduação**, especialmente em contextos de **Interação Humano–Robô (HRI)** e **avaliação de modelos de linguagem**.

---

## Objetivo do Projeto

O objetivo deste projeto é **demonstrar e justificar, de forma objetiva**, como a abordagem de **paráfrase baseada no modelo T5** pode ser validada para a geração de **dados sintéticos de comandos HRI**, garantindo:

- Preservação da intenção semântica;
- Diversidade linguística adequada;
- Contribuição para o treinamento robusto de modelos de reconhecimento de intenção.

A proposta enfatiza a **avaliação metodológica**, indo além do uso de métricas isoladas, e integrando análise quantitativa, interpretação qualitativa e princípios de reprodutibilidade científica.

---

## Escopo e Abordagem

O projeto explora, de forma controlada e ilustrativa:

- Uso de métricas linguísticas (BLEU, ROUGE);
- Avaliação da preservação de intenção via classificadores;
- Análise conjunta dos resultados para validação objetiva;
- Simulação de protocolos experimentais e geração de relatórios.

Os resultados apresentados são **simulados ou exemplificativos**, com o propósito de **explicitar o raciocínio científico**, e não de substituir experimentos empíricos completos.

---

## Observação Importante

Este repositório **não implementa experimentos reais em larga escala**, nem fornece modelos treinados para uso em ambientes de produção.  
Seu foco está na **estrutura conceitual**, **análise crítica** e **justificativa metodológica**, sendo apropriado para:

- Estudo individual;
- Atividades acadêmicas;
- Discussões metodológicas;
- Protótipos educacionais.

---

## Público-alvo

- Estudantes de graduação e pós-graduação;
- Pesquisadores em NLP, IA e HRI;
- Docentes e avaliadores acadêmicos;
- Desenvolvedores interessados em validação de dados sintéticos.

---

## Considerações Finais

Este projeto busca contribuir para a compreensão crítica do uso de dados sintéticos em NLP, enfatizando que **a validade de uma abordagem não depende apenas de resultados numéricos**, mas de uma **avaliação metodológica bem fundamentada**.
